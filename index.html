Recent Work

Recipe Nexus

Recipe Nexus is a web application for the easy access to great home style cooking recipes. This began as my senior project. The users can post recipes to the site, which eventually will yield a vast and diverse library of recipes. Users can then access the recipes anyone has created, as well as favorite them for future reference. The application utilizes amazon's Alexa, allowing users to use voice commands to control the web app, say in case of sticky hands. The application is being made using the python framework Django. Currently the application uses SQLite for its database while it is still in early production. To allow the application to communicate to Alexa amazon services such as Lambda, DynamoDB, and aws API Gateway were used. This project is still in production. Credit to my partner Mateusz Piekut. Check out the flyer. Check out the voice control architecture.
